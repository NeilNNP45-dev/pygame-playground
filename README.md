# Orbit-Sim 🌌

Orbit-Sim is a Python-based orbital mechanics simulator built using Pygame.

The project simulates celestial bodies moving under gravitational forces, allowing planets to orbit a central star in real time. It was created as a learning project to explore physics simulation, mathematical modeling, and object-oriented programming.

## Technologies Used

* Python
* Pygame
* Math module

## Run Locally

```bash
pip install pygame
python orbit_simulation.py
```
